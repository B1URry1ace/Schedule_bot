# Этап 1. Описание проекта и границы

1. Телеграм-бот с расписанием и заметками
2. Бот в телеграме, предоставляющий расписание занятий с возможностью оставлять заметки к выбранному занятию. Заметки можно
оставлять в текстовом виде или голосовым сообщением. Во втором случае распознаётся текст речи и форматируется в удобной формат
для чтения в текстовом виде.
3. Студент (в частности студент ЮФУ)
4. Предоставленеи расписания с заметками, добавление/изменение/удаление заметок. Выбор группы, автоматическое обновление расписания
5. Расписания по ФИО, расписание нескольких групп одновременно, видео-заметки
6. Telegram api, python, llm, whisper
