# Этап 2. Требования (MVP) + нефункциональные требования + критерии приёмки
## MUST
1. Выбор группы
2. Вывод расписания дня
3. Добавление/изменение/удаление заметок к конкретному занятию
4. Распознование речи в текст
5. Форматирования распознной речи в удобный для чтения текст заметки при помощи LLM
6. Сохранение заметок в базу данных
7. Настройки аккаунта/бота
8. Автоматическое обновление расписания
## NICE
1. Добавление фото к заметки
2. Корректный фильтр языковой модели
3. Распознавание занятия заметки прямо из голосового сообщения
